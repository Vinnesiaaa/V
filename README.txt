Vinnesia Static Website Guide
=============================

Website ini adalah versi HTML statis dari vinnesia.my.id, siap diupload ke hosting (Netlify, Vercel, cPanel, dst).

File Utama:
-----------
- index.html        : Homepage utama
- tools.html        : Daftar semua tools
- blog.html         : Daftar artikel blog
- currency.html     : Detail halaman Currency Tool
- qr.html           : Detail halaman QR Tool
- blog-currency.html: Artikel detail penggunaan Currency Tool
- blog-qr.html      : Artikel detail penggunaan QR Code

Fitur Tambahan:
---------------
- Desain profesional warna ungu kehitaman
- Mobile responsive
- SEO Meta Tags
- Favicon (gunakan file favicon.png di hosting Anda)

Upload:
-------
1. Extract ZIP
2. Upload semua file ke root public_html (jika cPanel) atau ke root project (Netlify)
3. Pastikan struktur file tetap
